# 이슈 트래커 시스템 백엔드

- 중앙대학교 2024 1학기 소프트웨어공학 01분반
- 13조 임채환, 김채원, 신두환, 이승종, 박지훈

### 주요 사용 의존성
- Spring Boot
- JPA & Spring Data JPA
- QueryDSL
- Lombok

### 프로젝트 실행 방법
```shell
// gradlew 실행 권한 부여
chmod 777 ./gradlew

// gradle 빌드
./gradlew clean build

// jar 파일 실행
java -jar build/libs/issue-tracker-server-0.0.1-SNAPSHOT.jar
```
